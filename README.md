# 🧠 Physio AI

An AI-powered physiotherapy assistant that helps users track posture, get exercise recommendations, and improve recovery using smart insights.

## 🚀 Features

* 🤖 AI Assistant for physiotherapy guidance
* 🧍 Body pain detection using interactive body map
* 📊 Progress tracking dashboard
* 🎥 Exercise video recommendations
* 🧘 Posture correction system

## 🛠️ Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* Supabase (Backend)
* TensorFlow.js (Posture detection)
